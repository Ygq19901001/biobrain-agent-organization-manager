# Agent Organization Manager 🏛️

**Multi-department agent org setup — hierarchy, communication protocols, reporting pipelines, learning systems, and audit frameworks.**

[![SkillHub](https://img.shields.io/badge/SkillHub-agent--org--manager-blue?style=flat)](https://skillhub.cn/skills/agent-org-manager)
![Version](https://img.shields.io/badge/version-1.0.0-green)
![License](https://img.shields.io/badge/license-MIT-lightgrey)

Don't just add agents — build an organization. With structure, communication, and accountability.

## What You Get

- **ROSTER Template** — Department definitions, role cards, capability matrices
- **Triple-Channel Communication** — sessions_send + shared files + broadcast — with anti-pattern checklist
- **Reporting Pipeline** — Daily departmental reporting with escalation paths
- **Learning System** — Scheduled department learning with topic rotation
- **Audit Framework** — Weekly inspection schedule with pass/fail criteria
